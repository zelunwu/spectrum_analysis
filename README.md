# Power Spectrum Density Analysis

PSD for time series.

## Functions

dft: Descrete Fourier Transform of time series.

asa: Autospectrum Analysis (Power Spectrum Density)

autocorr: Auto-Correlation Coefficient

cas: Cross Spectrum Analysis
