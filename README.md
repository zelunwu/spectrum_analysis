# Power Spectrum Density Analysis

PSD for time series.

## Functions

o_dft: Descrete Fourier Transform of time series.

o_asa: Autospectrum Analysis (Power Spectrum Density)

o_autocorr: Auto-Correlation Coefficient

o_cas: Cross Spectrum Analysis
